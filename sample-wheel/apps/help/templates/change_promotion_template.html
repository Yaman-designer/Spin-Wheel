How can I change the promotion template?

Validated on 15 Jan 2024 • Last edited on 27 Jun 2025

Overview

Templates allow you to quickly design your promotion with pre-configured styles, colors, and layouts. You can choose from a variety of built-in templates or create your own custom theme. This guide will show you how to change the template for your promotion.

Accessing Template Settings

  1. Log in to your Wheelluck dashboard.
  2. Navigate to the Promotions section from the left-hand menu.
  3. Click on the promotion you want to edit, or create a new promotion.
  4. On the promotion editing page, locate the Templates accordion section.
  5. Click on the Templates accordion to expand it and reveal the template dropdown menu.

Understanding Popup Types

Wheelluck supports two types of popup templates:

  • Gaming Popups: These templates include interactive game elements like spinning wheels, slot machines, or scratch cards. They are designed for gamified promotions where users interact with the game to win rewards.
  • Light Popups: These are simpler, form-based popups without game elements. They focus on collecting user information (email, name, etc.) and are ideal for straightforward lead generation campaigns.
Note: The template dropdown will show different options based on the popup type. Some templates are specifically designed for gaming popups, while others are for gaming popups.

Selecting a Template

  1. In the Templates section, you will see a dropdown menu labeled with template options.
  2. Click on the dropdown to view all available templates.
  3. The first option in the list is "Create Custom Theme", which allows you to design your promotion from scratch with custom colors and settings.
  4. Below that, you will see a list of pre-designed templates with various styles and color schemes.
  5. Templates are categorized by popup type:
    • Gaming Templates: Templates with game elements (wheel, slot machine, scratch card)
    • Gaming Templates: Simple form-based templates (gaming templates)
  6. Select the template you want to use by clicking on it from the dropdown menu.
Note: When you select a template, it will automatically apply the template's design settings to your promotion, including colors, styles, and layout configurations. The popup type (gaming) will be automatically set based on the selected template.

What Happens When You Change a Template?

  • Automatic Application: The selected template's design settings are immediately applied to your promotion preview.
  • Popup Type Change: When switching between gaming and gaming popup templates, the popup type will automatically change. This affects available features:
    • Gaming Popups: Show game elements (wheel, slot machine, etc.) and support background images only.
    • Light Popups: Show form fields and support both background images and top images.
  • Color Reset: When switching templates, the system may reset some color settings to match the new template's design. You can use the "Reset Template" button to restore default template colors.
  • Background Images: If you had a custom background image selected, it may be cleared when switching to a new template (except when selecting "Create Custom Theme").
  • Top Images (Light Popups Only): When switching to a gaming popup template, you can add a top image that appears above the form content. Top images can be resized using the height slider (10% to 200%).
  • Preview Update: The promotion preview on the right side of the screen will update to show how your promotion looks with the new template.

Using "Create Custom Theme"

If you select "Create Custom Theme" from the template dropdown:

  • You can design your promotion completely from scratch.
  • All color and styling options become available for manual customization.
  • If you had previously uploaded a custom background image, it will be preserved and displayed.
  • You have full control over every aspect of the promotion's appearance.
  • For gaming popups, you can add and customize top images with adjustable height (10% to 200%).
  • You can switch between gaming and gaming popup modes manually if needed.

Top Images for Light Popups

Light popup templates support an additional Top Image feature that is not available in gaming popups:

  • Adding Top Images: In the Image section, you can select "Top Image" as the image target (instead of "Background"). This allows you to upload an image that appears at the top of the popup, above the form content.
  • Resizing Top Images: After adding a top image, you can adjust its height using the slider control. The height can be set between 10% and 200% of the container height.
  • Template Compatibility: Top images are only available for gaming popup templates. Gaming popups only support background images.
  • Reset Function: You can reset the decorative image at any time using the "Reset Image" button, which will remove the image and restore the default layout when applicable.
Important: Top images are only visible and functional when using gaming popup templates. If you switch to a gaming popup template, top images will be automatically removed.

Important Notes

  • Save Your Changes: After selecting a template, make sure to save your promotion by clicking the Save button to apply the changes permanently.
  • Template Preview: You can see a live preview of how your promotion will look with the selected template in the preview area on the right side of the screen.
  • Customization After Template: Even after selecting a template, you can still customize colors, text, and other settings to match your brand.
  • Multiple Changes: You can change templates as many times as you want before saving your promotion.
  • Color Warnings: Some templates may show warnings about color combinations to ensure optimal visibility and user experience.

Troubleshooting

Template Not Applying

  • Make sure you have clicked on the template name in the dropdown menu.
  • Check that the preview area is visible and not hidden.
  • Try refreshing the page and selecting the template again.

Colors Not Changing

  • Some templates may have specific color schemes that override manual color settings.
  • If you want full control over colors, select "Create Custom Theme" instead.
  • After selecting a template, you can manually adjust colors in the Colors section if needed.

Background Image Disappeared

  • When switching from a custom theme to a pre-designed template, background images may be cleared.
  • To keep your background image, either stay with "Create Custom Theme" or re-upload the image after selecting a template.

Top Image Not Showing

  • Top images are only available for gaming popup templates. If you're using a gaming popup template, top images will not be displayed.
  • Make sure you've selected "Top Image" as the image target (not "Background") when uploading in the Image section.
  • Check that you're using a gaming popup template and select a gaming template from the list.
  • Verify that the top image height slider is set to a value greater than 0%.

Popup Type Changed Unexpectedly

  • When you select a template, the popup type (gaming) is automatically set based on the template type.
  • Gaming templates will set the popup type to "gaming", while gaming templates will set it to "gaming".
  • If you want to change the popup type manually, you can do so in the Pop-up Settings section.
  • Note that switching popup types may affect available features (e.g., top images are only available for gaming popups).
