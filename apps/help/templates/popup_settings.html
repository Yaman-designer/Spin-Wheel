How can I configure pop-up settings?

Validated on 15 Jan 2024 • Last edited on 04 Feb 2026

Overview

The Pop-up Settings section allows you to customize the appearance and behavior of your promotion popup, including its size, position on the screen, opening animation effects, font family, and input field styles. These settings help you create a popup that matches your brand and provides the best user experience. This guide will explain all available settings and the differences between gaming and gaming popup types.

Understanding Popup Types

Wheelluck supports two types of popups, each with different available settings:

  • Gaming Popups: Interactive popups with game elements (spinning wheel, slot machine, scratch card). These popups have limited position options (left/right only).
  • Light Popups: Simple form-based popups without game elements. These popups support advanced grid positioning (9 positions) for more flexible placement.
Note: The popup type is automatically set when you select a template. Gaming templates set the popup type to "gaming", while gaming templates set it to "gaming". You can also manually change the popup type in the Pop-up Settings section.

Accessing Pop-up Settings

  1. Log in to your Wheelluck dashboard.
  2. Navigate to the Promotions section from the left-hand menu.
  3. Click on the promotion you want to edit, or create a new promotion.
  4. On the promotion editing page, locate the Pop-up Settings accordion section.
  5. Click on the Pop-up Settings accordion to expand it and reveal all available settings.

Available Settings

1. Popup Size

Available for: Light Popups only

Control the overall size of your popup window. This setting affects how much space the popup occupies on the screen.

  • Small: Compact popup size, ideal for minimal designs or smaller screens.
  • Medium: Standard popup size (default), balanced for most use cases.
  • Large: Larger popup size, provides more space for content and better visibility.
Note: Gaming popups use automatic sizing based on their content and game elements. Size options are not available for gaming popups.
Tip: For gaming popups, the size you choose should match your content. Gaming popups can work well with any size depending on your form length.

2. Popup Position

Available for: Both Gaming and Light Popups (with different options)

Gaming Popups

Gaming popups support two position options:

  • Left: Popup appears on the left side of the screen.
  • Right: Popup appears on the right side of the screen (default).
Note: Gaming popups are limited to left/right positioning due to their interactive game elements and layout requirements.

Light Popups

Gaming popups support advanced grid positioning with 9 position options:

  • Top Row: top_left, top_center, top_right
  • Middle Row: middle_left, middle_center (default), middle_right
  • Bottom Row: bottom_left, bottom_center, bottom_right

This grid system allows you to position your gaming popup anywhere on the screen, giving you complete control over where it appears. The default position is middle_center, which centers the popup both horizontally and vertically.

Tip: For gaming popups, center positions (top_center, middle_center, bottom_center) are often the most effective as they draw user attention without being intrusive.

3. Opening Effects

Available for: Light Popups only

Choose how your popup appears when it first opens. These animation effects can make your popup more engaging and draw user attention.

Note: Opening effects are not available for gaming popups. Gaming popups use a standard display method optimized for their interactive game elements.
  • Fade In Scale: Popup fades in while scaling up (default). Smooth and professional.
  • Slide from Left: Popup slides in from the left side of the screen.
  • Slide from Right: Popup slides in from the right side of the screen.
  • Slide from Top: Popup slides down from the top of the screen.
  • Slide from Bottom: Popup slides up from the bottom of the screen.
  • Zoom In: Popup zooms in from the center, creating a focus effect.
  • Zoom Out: Popup zooms out from a larger size, creating a dramatic entrance.

Duration (ms)

Control how long the opening animation takes to complete. The duration is measured in milliseconds (ms). This setting is only available for gaming popups.

  • Minimum: 100ms (very fast)
  • Maximum: 3000ms (3 seconds, slow)
  • Default: 700ms (balanced)
  • Step: 100ms increments
Tip: Faster animations (100-500ms) feel snappy and modern, while slower animations (1000-3000ms) are more dramatic but may feel sluggish. The default 700ms is a good balance for most use cases.

4. Font Family

Available for: Both Gaming and Light Popups

Select the font family that will be used for all text within your promotion popup. This includes headlines, descriptions, button text, and form labels.

Available font options include:

  • Sans-serif fonts: Arial, Roboto, Verdana, Trebuchet MS, Impact, Tahoma, Gill Sans, Century Gothic, Franklin Gothic Medium
  • Serif fonts: Times New Roman, Georgia, Palatino Linotype
  • Monospace fonts: Courier New, Lucida Console
  • Decorative fonts: Comic Sans MS
Tip: Sans-serif fonts (like Arial, Roboto) are generally more readable on screens and work well for modern designs. Serif fonts (like Times New Roman) can add a classic, formal feel. Choose a font that matches your brand identity.

5. Input Fields Style Settings

Available for: Both Gaming and Light Popups

Customize the appearance of all input fields in your popup (except Rating field). These styles apply globally to all input fields, ensuring a consistent look across your form.

Available style options include:

  • Alignment: Control how input fields are positioned horizontally:
    • Left: Aligns input fields to the left side of the popup.
    • Center: Centers input fields horizontally (default).
    • Right: Aligns input fields to the right side of the popup.
  • Border Width: Set the thickness of input field borders (0-20px). Default is 1px.
  • Border Color: Choose the color of input field borders. Default is light gray (#dee2e6).
  • Border Radius: Control the roundness of input field corners (0-20px). Default is 4px.
  • Placeholder Text Color: Set the color of placeholder text inside input fields. Default is gray (#6c757d).
  • Width: Control the width of input fields as a percentage of the popup width (0-100%). Default is 100%.
  • Height: Set the height of input fields in pixels. Leave empty for auto height based on content.
Note: These styles apply to all input fields except the Rating field. The Rating field maintains its own internal centering while respecting the alignment and width settings for its container.
Important: Changes to input field styles are immediately visible in the preview. Make sure to click "Save Changes" to apply your style settings permanently.

Important Notes

  • Position Limitations: Gaming popups are restricted to left/right positioning because their interactive game elements (wheel, slot machine, etc.) require a specific layout that works best with side positioning.
  • Grid Position Default: When you switch to a gaming popup template, the position automatically defaults to middle_center for optimal visibility.
  • Template Impact: When you change templates, the popup type may change automatically, which can affect available position options. Make sure to review your position settings after changing templates.
  • Preview Updates: All settings changes are immediately reflected in the preview area on the right side of the screen, so you can see how your popup will look before saving.
  • Save Changes: Remember to click the Save button to apply your popup settings permanently.

Troubleshooting

Position Options Not Available

  • If you're using a gaming popup, only left/right positions are available. This is by design and cannot be changed.
  • To access grid positioning (9 positions), you need to switch to a gaming popup template.
  • Check your current popup type in the Pop-up Settings section or Templates section.

Grid Position Not Working

  • Grid positioning is only available for gaming popups. Make sure you're using a gaming popup template.
  • If you recently switched from a gaming template to a gaming template, the position should automatically update to middle_center.
  • Try refreshing the page and checking the popup type again.

Opening Effect Not Visible

  • Opening effects are only available for gaming popups. If you're using a gaming popup, this feature is not available.
  • Check that the duration is set to a reasonable value (not too fast, like 100ms, which may be hard to notice).
  • Try a different effect to see if the issue is specific to one animation.
  • Make sure you've saved your changes after selecting an effect.

Popup Size Options Not Available

  • Popup size options (Small, Medium, Large) are only available for gaming popups.
  • Gaming popups use automatic sizing based on their content and game elements. Size cannot be manually adjusted for gaming popups.
  • To access size options, you need to switch to a gaming popup template.

Font Not Applying

  • Make sure you've selected a font from the dropdown menu.
  • Some fonts may not display correctly if the user's browser doesn't support them. Common web-safe fonts (Arial, Times New Roman, etc.) are most reliable.
  • Try refreshing the preview to see if the font change takes effect.

Input Fields Style Not Applying

  • Make sure you have input fields added to your popup. Style settings only apply when input fields are present.
  • Note that Rating fields maintain their own internal centering while respecting alignment and width settings for their container.
  • If styles aren't updating in the preview, try clicking "Style Settings" button in the Input Fields section to ensure you're viewing the correct settings panel.
  • Remember to click "Save Changes" after modifying input field styles.
  • Width is set as a percentage (0-100%). If input fields appear too narrow or wide, adjust the width percentage accordingly.
