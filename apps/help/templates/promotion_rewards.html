How can I configure promotion rewards?

Validated on 15 Jan 2024 • Last edited on 27 Jun 2025

Overview

The Rewards section allows you to configure the prizes, discounts, or outcomes that users can win when they spin the wheel. You can set up to 12 different rewards, each with a label (what users see), a coupon code (for winning rewards), and a weight (win probability). This guide will show you how to configure rewards and understand how Smart Reward (AI-powered triggers) affects reward management.

Accessing Rewards Settings

  1. Log in to your Wheelluck dashboard.
  2. Navigate to the Promotions section from the left-hand menu.
  3. Click on the promotion you want to edit, or create a new promotion.
  4. On the promotion editing page, locate the Rewards accordion section.
  5. Click on the Rewards accordion to expand it and reveal all reward configuration fields.

Understanding Reward Structure

Your promotion has 12 reward slots, arranged in an alternating pattern:

  • Win Rewards (Editable): Positions 1, 3, 5, 7, 9, and 11 are "win" rewards that users can actually win. These have editable labels, coupon codes, and weights.
  • No-Win Rewards (Locked): Positions 2, 4, 6, 8, 10, and 12 are "no-win" messages (like "Sorry", "Almost", "Next time"). These are locked and cannot be edited when Smart Reward is disabled.
Note: When Smart Reward is disabled, only the win rewards (positions 1, 3, 5, 7, 9, 11) can be edited. When Smart Reward is enabled, all rewards become locked and are managed by the AI system.

Reward Fields Explained

1. Label

Purpose: The text that appears on the wheel slice that users see when spinning.

  • This is what users see on the wheel before and during the spin.
  • Maximum length: 13 characters
  • This field is required for all rewards.
  • Examples: "50% OFF", "Free Shipping", "20% OFF", "Try Again"
Tip: Keep labels short and clear since they appear on the wheel. Use abbreviations if needed (e.g., "50% OFF" instead of "Fifty Percent Off").

2. Coupon Code

Purpose: The actual discount code or coupon that users receive when they win that reward.

  • This code is displayed to users after they win and can be used during checkout.
  • Maximum length: 10 characters
  • Required for win rewards only (positions 1, 3, 5, 7, 9, 11).
  • No-win rewards (positions 2, 4, 6, 8, 10, 12) don't need coupon codes and the field is disabled.
  • Examples: "SAVE50", "FREESHIP", "WELCOME20"
Important: Win rewards (positions 1, 3, 5, 7, 9, 11) must have a coupon code. If you leave it blank, you'll receive a validation error when saving.

3. Weight (%)

Purpose: Determines the probability or chance that a user will win that specific reward.

  • Weight is expressed as a percentage (0-100).
  • Higher weight = higher chance of winning that reward.
  • Maximum value: 99 (2 digits maximum)
  • Weight cannot be negative.
  • When Smart Reward is disabled, the total of all weights must equal 100%.
  • When Smart Reward is enabled, weights are managed by AI and don't need to total 100%.
Example: If Reward 1 has weight 30 and Reward 3 has weight 70, users have a 30% chance of winning Reward 1 and a 70% chance of winning Reward 3 (assuming only these two have weights).

Configuring Rewards (Smart Reward Disabled)

When Smart Reward (AI-powered triggers) is disabled, you have manual control over rewards:

  1. Only win rewards (positions 1, 3, 5, 7, 9, 11) can be edited.
  2. No-win rewards (positions 2, 4, 6, 8, 10, 12) are locked and cannot be changed.
  3. For each win reward, fill in:
    • Label: What appears on the wheel (required, max 13 characters)
    • Coupon Code: The discount code users receive (required, max 10 characters)
    • Weight: Win probability percentage (required, 0-99)
  4. Total Weight Must Equal 100%: The sum of all reward weights must equal exactly 100. For example:
    • Reward 1: 30%
    • Reward 3: 50%
    • Reward 5: 20%
    • Total: 100% ✓
  5. You can use the weight calculator buttons (if available) to help distribute weights automatically.
  6. Click Save to save your reward configuration.
Validation Rule: If the total weight doesn't equal 100%, you'll receive an error: "The total weight of the rewards must be 100". You must adjust weights until they total exactly 100.

Smart Reward (AI-Powered Triggers) - Overview

Smart Reward is an AI-powered feature that uses Google Analytics data to intelligently manage reward distribution. When enabled, the AI system analyzes user behavior and determines optimal reward timing and distribution.

How It Works: When Google Analytics is available on your site, Smart Reward can use traffic and behavior signals to decide when to show promotions and which rewards to offer, with the goal of improving conversion and engagement.

What Happens When Smart Reward Is Enabled?

When you enable "AI powered triggers" (Smart Reward) in the Options section:

1. All Reward Fields Become Locked

  • All 12 rewards (both win and no-win) become disabled and cannot be edited.
  • Label, Coupon Code, and Weight fields for all rewards are locked.
  • You cannot manually change reward labels, codes, or weights.
  • The AI system manages all reward configurations automatically.

2. Weight Requirements Change

  • Total weight no longer needs to equal 100%.
  • The AI system manages weight distribution intelligently.
  • You don't need to manually calculate or adjust weights.
  • All weight values are set to 0 and managed by the AI.

3. Google Analytics Property ID Required

  • A Google Analytics Property ID field appears and becomes required.
  • You must enter your Google Analytics Property ID (format: UA-123456789-1 or G-XXXXXXXXXX).
  • This ID is used to connect your promotion to Google Analytics data.
  • If you leave this field empty, you'll receive an error: "If the Ai powered triggers is active, the property id field cannot be left empty".

4. Weight Calculator Disappears

  • The weight calculator buttons (if available) are hidden.
  • Manual weight calculation tools are no longer needed since AI manages weights.

5. Analytics Active Message Appears

  • An "Analytics Active" message or indicator appears to confirm Smart Reward is enabled.
  • This confirms that the AI system is using Google Analytics data.

6. Some Options Become Disabled

  • "Trigger when the user intends to leave" is automatically disabled and unchecked.
  • "Trigger when scroll down" is automatically disabled and unchecked.
  • "Trigger with delay" is automatically disabled, unchecked, and cleared.
  • These options are disabled because Smart Reward uses AI to determine optimal trigger timing instead.

7. Reward Distribution Becomes Intelligent

  • The AI analyzes user behavior patterns from Google Analytics.
  • Reward distribution is optimized based on user data and conversion patterns.
  • The system makes data-driven decisions about which rewards to offer and when.
  • This can lead to better conversion rates and more effective promotions.
Benefits of Smart Reward: By using AI and Google Analytics data, Smart Reward can optimize your promotion performance, improve conversion rates, and provide better user experiences through intelligent, data-driven reward management.

Enabling Smart Reward

To enable Smart Reward:

  1. Go to the Options section in your promotion editor.
  2. Find the "AI powered triggers" checkbox.
  3. Click the checkbox to enable Smart Reward.
  4. The Google Analytics Property ID field will appear.
  5. Enter your Google Analytics Property ID (you can find this in your Google Analytics account).
  6. All reward fields will become locked automatically.
  7. Click Save to save your changes.
Important: Once Smart Reward is enabled, you cannot manually edit rewards. If you need to change rewards, you must first disable Smart Reward, make your changes, then re-enable it (if desired).

Disabling Smart Reward

To disable Smart Reward and regain manual control:

  1. Go to the Options section.
  2. Uncheck the "AI powered triggers" checkbox.
  3. All reward fields will become editable again (win rewards only).
  4. The system may automatically assign random weights that total 100%.
  5. You can then manually edit reward labels, codes, and weights.
  6. Make sure the total weight equals 100% before saving.
  7. Click Save to save your changes.
Automatic Weight Assignment: When you disable Smart Reward, the system may automatically assign random weights to help you get started. You can then adjust these weights manually to total 100%.

Validation Rules

The following validation rules apply when configuring rewards:

General Rules (Apply to All Rewards)

  • Label Required: All rewards must have a label. Empty labels are not allowed.
  • Weight Required: All rewards must have a weight value. Empty weights are not allowed.
  • Weight Must Be a Number: Weight values must be valid numbers (0-99).
  • Weight Cannot Be Negative: Weight values cannot be negative numbers.

Win Reward Rules (Positions 1, 3, 5, 7, 9, 11)

  • Coupon Code Required: Win rewards must have a coupon code. Empty coupon codes are not allowed for these positions.
  • Code Maximum Length: Coupon codes cannot exceed 10 characters.

Weight Total Rules

  • When Smart Reward is Disabled: The total of all reward weights must equal exactly 100%. If it doesn't, you'll receive an error: "The total weight of the rewards must be 100".
  • When Smart Reward is Enabled: Total weight does not need to equal 100%. The AI system manages weight distribution automatically.
Validation Errors: If any of these rules are violated, you'll see specific error messages indicating what needs to be corrected before you can save your promotion.

Best Practices

For Manual Reward Configuration (Smart Reward Disabled)

  • Balance Your Weights: Distribute weights strategically. Higher-value rewards (like 50% OFF) should typically have lower weights, while lower-value rewards can have higher weights.
  • Test Different Distributions: Experiment with different weight combinations to find what works best for your business goals.
  • Keep Labels Short: Since labels are limited to 13 characters, use concise, clear text that users can quickly understand.
  • Use Meaningful Coupon Codes: Create coupon codes that are easy to remember and relate to your brand or campaign.
  • Verify Total Weight: Always double-check that your weights total exactly 100% before saving.

For Smart Reward (AI-Powered)

  • Ensure Google Analytics is Set Up: Make sure you have Google Analytics properly configured on your website before enabling Smart Reward.
  • Use Correct Property ID: Double-check that you're entering the correct Google Analytics Property ID. You can find this in your Google Analytics account under Admin → Property Settings.
  • Allow Time for Learning: The AI system needs time to analyze data and learn patterns. Allow the system to run for a while to see optimal results.
  • Monitor Performance: Keep an eye on your promotion performance metrics to see how Smart Reward is affecting conversions.

Important Notes

  • Reward Positions: Remember that positions 1, 3, 5, 7, 9, 11 are win rewards (editable when Smart Reward is off), while positions 2, 4, 6, 8, 10, 12 are no-win messages (always locked when Smart Reward is off).
  • Smart Reward Lock: When Smart Reward is enabled, you cannot edit any rewards. You must disable Smart Reward first to make changes.
  • Weight Total: When Smart Reward is disabled, weights must total exactly 100%. Use a calculator if needed to verify.
  • Coupon Code Requirements: Only win rewards (positions 1, 3, 5, 7, 9, 11) require coupon codes. No-win rewards don't need codes.
  • Save Your Changes: Always remember to click the Save button after configuring rewards to ensure the changes are saved permanently.
  • Preview Updates: Changes to reward labels are immediately reflected in the preview panel, so you can see how they look on the wheel.

Troubleshooting

"The total weight of the rewards must be 100" Error

  • This error appears when Smart Reward is disabled and your total weight doesn't equal 100%.
  • Check the sum of all reward weights. They must total exactly 100.
  • Adjust individual weights until the total equals 100.
  • Use the weight calculator buttons (if available) to help distribute weights automatically.
  • Alternatively, enable Smart Reward if you want the AI to manage weights automatically.

"The rewards label cannot be blank" Error

  • One or more rewards have an empty label field.
  • All rewards must have a label, even if it's just a placeholder.
  • Fill in labels for all rewards that are showing this error.

"The rewards weight cannot be blank" Error

  • One or more rewards have an empty weight field.
  • All rewards must have a weight value (can be 0 for no-win rewards).
  • Enter a weight value (0-99) for all rewards.

"The rewards weight cannot be negative" Error

  • You've entered a negative number in a weight field.
  • Weight values must be 0 or positive numbers.
  • Change any negative weights to 0 or a positive number.

"The rewards weight must be a valid number" Error

  • You've entered non-numeric text in a weight field.
  • Weight values must be numbers only (0-99).
  • Remove any letters or special characters from weight fields.

"The rewards coupon code cannot be blank" Error

  • One of the win rewards (positions 1, 3, 5, 7, 9, 11) has an empty coupon code.
  • Win rewards require coupon codes.
  • Enter a coupon code (max 10 characters) for all win rewards.

Cannot Edit Rewards (All Fields Disabled)

  • Check if Smart Reward (AI powered triggers) is enabled in the Options section.
  • When Smart Reward is enabled, all reward fields are locked.
  • Disable Smart Reward in the Options section to regain manual control over rewards.

Cannot Edit Specific Rewards (Some Fields Disabled)

  • When Smart Reward is disabled, only win rewards (positions 1, 3, 5, 7, 9, 11) can be edited.
  • No-win rewards (positions 2, 4, 6, 8, 10, 12) are always locked when Smart Reward is off.
  • This is normal behavior - only win rewards are editable in manual mode.

Weight Calculator Buttons Not Visible

  • Check if Smart Reward is enabled. Weight calculator is hidden when Smart Reward is active.
  • Disable Smart Reward to see and use the weight calculator buttons.
  • The calculator helps automatically distribute weights to total 100%.

Cannot Enter More Than 2 Digits in Weight Field

  • Weight fields are limited to 2 digits (0-99).
  • This is a system limitation to keep weights manageable.
  • If you need higher percentages, distribute them across multiple rewards.

Label or Code Text Gets Cut Off

  • Labels are limited to 13 characters and codes to 10 characters.
  • If your text is too long, it will be truncated.
  • Use shorter, more concise text that fits within the limits.
  • Consider using abbreviations for labels if needed.
