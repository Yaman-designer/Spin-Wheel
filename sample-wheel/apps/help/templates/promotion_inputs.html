How can I add custom input fields to my promotion?

Validated on 15 Jan 2024 • Last edited on 27 Jun 2025

Overview

The Input section allows you to add custom input fields to your promotion form, in addition to the default email field. This feature enables you to collect additional information from users such as their full name, phone number, company name, or birth date. These custom fields appear in your promotion popup below the email input field, allowing you to gather more detailed information from participants. This guide will show you how to add, manage, and remove custom input fields.

Accessing Input Settings

  1. Log in to your Wheelluck dashboard.
  2. Navigate to the Promotions section from the left-hand menu.
  3. Click on the promotion you want to edit, or create a new promotion.
  4. On the promotion editing page, locate the Input accordion section.
  5. Click on the Input accordion to expand it and reveal the input field management options.

Understanding the Input Section

1. Email Field (Required)

Purpose: The default email input field that is always present in your promotion form.

  • This field is required and cannot be removed or modified.
  • It appears at the top of the input fields section in your promotion popup.
  • Users must enter a valid email address to participate in the promotion.
  • The field is displayed as read-only in the editor to indicate it cannot be changed.
Note: The email field is essential for tracking participants and sending rewards, so it cannot be disabled or removed.

2. Additional Input Fields

Purpose: Custom input fields that you can add to collect additional information from users.

  • These fields appear below the email field in your promotion popup.
  • You can add multiple custom input fields to your form.
  • Each field type has a predefined format and validation.
  • Fields are saved to your database along with the email address when users participate.

Available Input Field Types

You can choose from the following predefined input field types:

1. Full Name

  • Type: Text input
  • Placeholder: "Enter your full name"
  • Purpose: Collects the user's complete name (first and last name).
  • Use Case: Personalize communications, create mailing lists, or for personalized rewards.

2. Phone Number

  • Type: Telephone input
  • Placeholder: "Enter your phone number"
  • Purpose: Collects the user's phone number for SMS marketing or direct contact.
  • Use Case: SMS campaigns, customer support, or phone-based verification.

3. Company Name

  • Type: Text input
  • Placeholder: "Enter your company name"
  • Purpose: Collects the user's company or business name.
  • Use Case: B2B marketing, lead qualification, or business-focused promotions.

4. Birth Date

  • Type: Date input
  • Placeholder: "Select your birth date"
  • Purpose: Collects the user's date of birth.
  • Use Case: Birthday campaigns, age verification, or personalized birthday offers.

How to Add Input Fields

  1. Open the Input section in your promotion editor.
  2. Locate the "Additional Input Fields" dropdown menu.
  3. Click on the dropdown to see available input field types.
  4. Select the input field type you want to add (Full Name, Phone Number, Company Name, or Birth Date).
  5. The selected field will immediately appear in the list below the dropdown.
  6. You can add multiple fields by repeating this process.
  7. Each field you add will appear in your promotion popup below the email field.
  8. Click Save Changes to save your input field configuration.
Duplicate Prevention: You cannot add the same input field type twice. If you try to add a field that already exists, you'll see a message indicating it has already been added.

How to Remove Input Fields

  1. In the Input section, locate the input field you want to remove.
  2. Each added field displays its label (e.g., "Full Name", "Phone Number") with a red "X" button on the right.
  3. Click the red "X" button next to the field you want to remove.
  4. The field will be immediately removed from the list.
  5. The field will also disappear from the preview.
  6. Click Save Changes to permanently save the removal.
Note: Removing a field updates the form from that point on. This product does not keep a server-side archive of past form submissions tied to the field list.

Field Display Order

Input fields appear in your promotion popup in the following order:

  1. Email Field (always first, cannot be moved or removed)
  2. Custom Input Fields (appear in the order you added them, below the email field)
Display: All input fields appear in a clean, consistent format matching the email field's style. They are displayed vertically, one below the other, in your promotion popup.

Data collection

The editor configures what visitors see in the popup. Per-field values and email addresses are not stored in this application for later viewing or export.

Best Practices

Choosing the Right Fields

  • Keep It Relevant: Only add fields that are relevant to your promotion and marketing goals.
  • Don't Overwhelm: Too many fields can reduce conversion rates. Keep it to 2-3 additional fields maximum.
  • Consider Privacy: Be mindful of asking for sensitive information like birth dates. Only collect what you need.
  • Match Your Goals: Choose fields that align with your marketing strategy (e.g., phone for SMS campaigns, company name for B2B).

Field Order Strategy

  • Start Simple: Add the most important fields first (e.g., Full Name before Company Name).
  • Logical Flow: Order fields in a way that makes sense to users (e.g., Name → Phone → Company).
  • Test Different Combinations: Experiment with different field combinations to see what works best for your audience.

Conversion Optimization

  • Progressive Disclosure: Start with just email, then gradually add fields based on user engagement.
  • Value Exchange: Make sure the value of your promotion justifies asking for additional information.
  • Clear Purpose: Users are more likely to provide information if they understand why you need it.

Preview and Testing

Before publishing your promotion:

  1. Use the preview panel on the left side of the editor to see how your input fields appear.
  2. Check that all fields are visible and properly formatted.
  3. Verify the field order matches your expectations.
  4. Test the form on both desktop and mobile views using the preview toggle buttons.
  5. Ensure placeholder text is clear and helpful for users.
Live Preview: Changes to input fields are immediately reflected in the preview panel, so you can see how they look in real-time.

Important Notes

  • Email Field: The email field is always required and cannot be removed or modified.
  • No Duplicates: You cannot add the same input field type more than once.
  • Save Changes: Always click Save Changes after adding or removing fields to ensure your configuration is saved.
  • Field list: Removing a field only changes the live form configuration going forward.
  • Field Validation: Each field type has built-in validation (e.g., email format, date format, phone format).
  • Mobile Compatibility: All input fields are fully responsive and work on mobile devices.
  • Required Fields: Currently, only the email field is required. Additional fields are optional for users.

Troubleshooting

Cannot Add Input Field - "Already Added" Message

  • You're trying to add an input field type that has already been added.
  • Each field type can only be added once per promotion.
  • Check the list of added fields to see which ones are already present.
  • If you need that field type, you must first remove it and then add it again.

Input Field Not Appearing in Preview

  • Make sure you've clicked Save Changes after adding the field.
  • Refresh the preview panel or the entire page.
  • Check that the field was actually added to the list (it should appear below the dropdown).
  • Verify you're looking at the correct preview (desktop or mobile).

Cannot Remove Input Field

  • Make sure you're clicking the red "X" button on the correct field.
  • Check that the field is actually in the list (it should be visible below the dropdown).
  • Try refreshing the page and attempting to remove it again.

Fields Not Saving

  • Make sure you've clicked the Save Changes button at the bottom of the page.
  • Wait for the success message confirming your changes were saved.
  • Check for any validation errors that might prevent saving.
  • Try refreshing the page and adding the fields again.

Fields Not Appearing in Live Promotion

  • Verify that you've saved your changes after adding the fields.
  • Check that your promotion is active and published.
  • Clear your browser cache and test again.
  • Verify the promotion code is correctly installed on your website.
  • Check that you're testing on the correct website URL where the promotion is installed.

Field data not visible after submit

  • Verify that the fields are visible in your live promotion popup.
  • Check that users are actually filling out the fields before submitting.
  • This app does not provide a stored log of submissions; use your own integrations or site tooling if you need that.
  • Ensure your promotion is active and receiving traffic.
