How can I change colors in my promotion theme?

Validated on 15 Jan 2024 • Last edited on 27 Jun 2025

Overview

Customizing colors is an important part of creating a promotion that matches your brand. However, the ability to change colors depends on which template you're using. This guide explains how to change colors and which elements can be customized in different templates.

Understanding Template Restrictions

Important: When you select a pre-designed template (anything other than "Create Custom Theme"), some color customization options are automatically disabled to maintain the template's design integrity.

  • "Create Custom Theme" (Default Template): All colors can be customized freely.
  • Pre-designed Templates: Only certain colors can be changed. Other colors are locked to preserve the template's design.

Accessing Color Settings

  1. Log in to your Wheelluck dashboard.
  2. Navigate to the Promotions section from the left-hand menu.
  3. Click on the promotion you want to edit, or create a new promotion.
  4. On the promotion editing page, locate the Colors accordion section.
  5. Click on the Colors accordion to expand it and reveal all color customization options.

Color Elements Available

The Colors section includes the following elements that can be customized:

  • Container: Background color and text color for the main popup container.
  • Button: Background color and text color for the spin button.
  • Slice #1, #2, #3, #4: Background colors and text colors for each wheel slice segment.
  • Circle: Background color and text color for the center circle of the wheel.
  • Pin: Background color and text color for the wheel pointer/pin.

Changing Colors with "Create Custom Theme"

When you have selected "Create Custom Theme" from the Templates dropdown:

  1. All color boxes in the Colors section are fully enabled and clickable.
  2. Click on any color box to open the color picker.
  3. Select your desired color from the color picker.
  4. The color will be immediately applied to the preview on the right side of the screen.
  5. You can change any color element: Container, Button, Slices, Circle, and Pin.
  6. Both background colors and text colors can be customized for each element.
Full Control: With "Create Custom Theme", you have complete freedom to customize every color element to match your brand identity.

Changing Colors with Pre-designed Templates

When you select a pre-designed template (any template other than "Create Custom Theme"):

Colors That CAN Be Changed:

  • Container Text Color: The text color inside the main popup container.
  • Button Background Color: The background color of the spin button.
  • Button Text Color: The text color on the spin button.
  • Pin Background Color: The background color of the wheel pointer.
  • Pin Text Color: The text color on the wheel pointer.

Colors That CANNOT Be Changed:

  • Container Background Color: Locked to maintain template design.
  • Slice Background Colors (Slice #1, #2, #3, #4): Locked to preserve the template's color scheme.
  • Slice Text Colors: Locked to ensure readability with the template's design.
  • Circle Background Color: Locked to maintain template consistency.
  • Circle Text Color: Locked to preserve template design.
Visual Indicators: When a color cannot be changed, the color box will appear grayed out (50% opacity) and show a tooltip message: "To change this element's color, please use the Default Template." when you hover over it.

How to Change a Color

  1. Make sure the color box is enabled (not grayed out).
  2. Click on the color box next to the element you want to customize.
  3. A color picker will appear.
  4. Select your desired color by:
    • Clicking on a color in the color palette.
    • Entering a hex color code directly (e.g., #FF5733).
    • Adjusting the color using the color picker controls.
  5. The color will be immediately applied to the preview.
  6. Click Save to save your promotion with the new colors.

Why Some Colors Are Locked

Pre-designed templates have locked colors for several important reasons:

  • Design Integrity: Templates are carefully designed with specific color combinations that work well together. Allowing all colors to be changed could result in poor contrast or unreadable text.
  • Visual Consistency: The template's visual identity is maintained by keeping core colors consistent.
  • User Experience: Locked colors ensure that the promotion remains visually appealing and professional-looking.
  • Readability: Text colors are locked to ensure they remain readable against their background colors.
Tip: If you need full control over all colors, switch to "Create Custom Theme" from the Templates dropdown. This will unlock all color customization options.

Switching Templates to Unlock Colors

If you want to customize colors that are currently locked:

  1. Go to the Templates section in the promotion editor.
  2. Click on the template dropdown menu.
  3. Select "Create Custom Theme" from the list.
  4. All color boxes will now be enabled and clickable.
  5. You can now customize any color element as needed.
  6. Note: When switching to "Create Custom Theme", the template's default colors will be applied, but you can then change them all.
Important: When you switch from a pre-designed template to "Create Custom Theme", some design elements may change. Make sure to review your promotion preview after switching templates.

Important Notes

  • Save Your Changes: Always remember to click the Save button after making color changes to ensure they are saved permanently.
  • Preview First: Use the preview panel on the right side of the screen to see how your color changes look before saving.
  • Color Contrast: When using "Create Custom Theme", ensure there is sufficient contrast between text colors and background colors for readability.
  • Brand Consistency: Choose colors that match your brand identity and website design.
  • Background Images: Note that background image uploads are also only available when using "Create Custom Theme".

Troubleshooting

Color Box Is Grayed Out and Not Clickable

  • This means you're using a pre-designed template and that specific color cannot be changed.
  • Hover over the grayed-out color box to see a tooltip explaining why it's disabled.
  • To change this color, switch to "Create Custom Theme" in the Templates section.

Color Changes Not Appearing in Preview

  • Make sure you clicked on the color box and selected a color from the color picker.
  • Check that the color box is enabled (not grayed out).
  • Try refreshing the preview or clicking on another element and then back to the color box.
  • Ensure you're not trying to change a locked color in a pre-designed template.

All Colors Are Locked

  • Check which template is currently selected in the Templates dropdown.
  • If you're using a pre-designed template, only certain colors (Container Text, Button, and Pin colors) can be changed.
  • To unlock all colors, select "Create Custom Theme" from the Templates dropdown.

Color Picker Not Opening

  • Make sure you're clicking directly on the color box, not the surrounding area.
  • Check if the color box is enabled (not grayed out).
  • Try refreshing the page and attempting again.
  • Ensure you have JavaScript enabled in your browser.
