How can I configure the countdown timer?

Validated on 15 Jan 2024 • Last edited on 27 Jun 2025

Overview

The Countdown feature adds a timer that displays how much time remains for users to use their winning coupon code. This creates urgency and encourages users to complete their purchase before the coupon expires. The countdown appears after a user wins a reward and shows the remaining time in minutes and seconds. This guide will show you how to enable, configure, and customize the countdown timer.

Accessing Countdown Settings

  1. Log in to your Wheelluck dashboard.
  2. Navigate to the Promotions section from the left-hand menu.
  3. Click on the promotion you want to edit, or create a new promotion.
  4. On the promotion editing page, locate the Countdown accordion section.
  5. Click on the Countdown accordion to expand it and reveal countdown configuration options.

Understanding the Countdown Timer

The countdown timer serves an important purpose in your promotion:

  • When It Appears: The countdown appears after a user successfully spins the wheel and wins a reward (coupon code).
  • What It Shows: It displays the remaining time (in minutes and seconds) that the user has to use their coupon code before it expires.
  • Where It Appears: The countdown is displayed in the coupon code panel that appears after a win, typically showing the format "MM:SS" (minutes:seconds) or "HH:MM:SS" (hours:minutes:seconds) if the time is longer than an hour.
  • Purpose: Creates urgency, encourages immediate action, and helps users understand they have limited time to use their reward.
User Experience: The countdown helps convert winners into customers by creating a sense of urgency. Users see the time ticking down and are more likely to complete their purchase quickly.

Enabling the Countdown

To enable the countdown timer:

  1. In the Countdown section, you'll see a toggle switch labeled "Display countdown reward reminder".
  2. Click on the toggle switch to enable the countdown.
  3. Once enabled, the Valid Time (Minutes) field will become required.
  4. Enter the number of minutes you want the countdown to run (e.g., 15 for 15 minutes).
  5. Configure the countdown colors (background and text) if desired.
  6. Click Save to save your countdown settings.
Live Preview: When you enable the countdown, you'll see it appear in the preview panel, showing how it will look to users.

Setting Valid Time (Minutes)

The Valid Time determines how long users have to use their coupon code:

  • Purpose: Sets the duration (in minutes) that the coupon code will be valid after a user wins.
  • Required When Active: If countdown is enabled, this field is required and cannot be left empty.
  • Format: Enter a number representing minutes (e.g., 15 for 15 minutes, 30 for 30 minutes, 60 for 1 hour).
  • Validation: The value must be a valid number and cannot be negative.
  • Default: The default value is typically 15 minutes.
Best Practice: Common valid times are 15, 30, or 60 minutes. Shorter times (15 minutes) create more urgency, while longer times (60+ minutes) give users more flexibility. Choose based on your business needs and typical checkout time.
Important: The valid time is entered in minutes, not seconds or hours. For example, enter "15" for 15 minutes, not "900" (which would be 900 minutes).

Customizing Countdown Colors

You can customize the appearance of the countdown timer by setting its colors:

Countdown Background Color

  • This is the background color of the countdown timer display.
  • Click on the color box to open the color picker.
  • Select your desired background color.
  • The default colors are a light mint background (#cee8e8) and teal text (#2C7A7B).

Countdown Text Color

  • This is the color of the countdown numbers/text.
  • Click on the color box to open the color picker.
  • Select your desired text color.
  • The default text color is typically black (#000000) or white (#ffffff).
Contrast Tip: Ensure there's good contrast between the background and text colors so the countdown is easily readable. For example, use dark text on a light background, or light text on a dark background.

How the Countdown Works

Once configured, here's how the countdown functions:

  1. User Wins: When a user spins the wheel and wins a reward (coupon code), the countdown timer is activated.
  2. Timer Starts: The countdown begins immediately, starting from the valid time you set (e.g., 15:00 for 15 minutes).
  3. Display Format: The countdown displays in "MM:SS" format (e.g., "15:00", "14:59", "14:58") for times under an hour, or "HH:MM:SS" format for longer durations.
  4. Countdown Decreases: The timer counts down second by second, showing the remaining time.
  5. Expiration: When the countdown reaches "00:00", the coupon code expires and can no longer be used.
  6. Persistence: The countdown time is stored in the user's browser, so it continues counting even if they refresh the page or navigate away and return.
Technical Note: The countdown uses browser localStorage to persist the expiration time, ensuring the timer continues accurately even if the user closes and reopens the page.

Disabling the Countdown

If you want to disable the countdown timer:

  1. Go to the Countdown section in your promotion editor.
  2. Click on the "Display countdown reward reminder" toggle to turn it off.
  3. The countdown will no longer appear after users win rewards.
  4. The Valid Time field becomes optional when countdown is disabled.
  5. Click Save to save your changes.
Note: When the countdown is disabled, users will still receive their coupon codes, but there will be no visible timer showing expiration time. The codes may still have expiration times set by your system, but users won't see a countdown.

Validation Rules

The following validation rules apply when configuring the countdown:

  • Valid Time Required: If "Display countdown reward reminder" is enabled, the Valid Time field is required and cannot be left empty.
  • Valid Number: The Valid Time must be a valid number. Non-numeric values (like letters or special characters) are not allowed.
  • Cannot Be Negative: The Valid Time cannot be a negative number. It must be 0 or a positive number.
  • Zero is Allowed: You can set the valid time to 0, though this would make the coupon expire immediately (not recommended).
Validation Errors: If any of these rules are violated, you'll see specific error messages:
  • "If the countdown reminder is active, the valid time field cannot be left blank"
  • "The valid time must be a valid number"
  • "The valid time cannot be a negative number"

Best Practices

  • Choose Appropriate Time: Set a valid time that gives users enough time to complete their purchase but creates urgency. 15-30 minutes is often a good balance.
  • Consider Checkout Time: Think about how long it typically takes users to complete checkout on your site. Set the countdown to be slightly longer than average checkout time.
  • Test Different Durations: Experiment with different valid times to see what works best for your conversion rates.
  • Color Contrast: Ensure the countdown colors have good contrast for readability. Test on different devices and screen sizes.
  • Brand Consistency: Choose countdown colors that match your brand colors and overall promotion design.
  • Mobile Visibility: Make sure the countdown is clearly visible on mobile devices, as many users will view it on phones.
  • Clear Communication: Consider adding text near the countdown explaining what it means (e.g., "Use your code within the time shown").

Important Notes

  • Time Format: Valid Time is entered in minutes, not seconds or hours. The system converts this to the appropriate display format automatically.
  • Required When Active: When countdown is enabled, the Valid Time field becomes required. You cannot save without entering a value.
  • Save Your Changes: Always remember to click the Save button after configuring countdown settings to ensure the changes are saved permanently.
  • Live Preview: Changes to countdown settings are immediately reflected in the preview panel, so you can see how it looks before saving.
  • Browser Storage: The countdown uses browser localStorage to persist the expiration time, ensuring accuracy across page refreshes.
  • Multiple Users: Each user who wins gets their own independent countdown timer that starts when they win.
  • Expiration Behavior: When the countdown reaches zero, the coupon code expires. Make sure users understand this before the timer expires.

Troubleshooting

"If the countdown reminder is active, the valid time field cannot be left blank" Error

  • You've enabled "Display countdown reward reminder" but haven't entered a valid time value.
  • When countdown is enabled, the Valid Time field is required.
  • Enter a number (in minutes) in the Valid Time field.
  • Alternatively, disable the countdown if you don't want to use it.

"The valid time must be a valid number" Error

  • You've entered non-numeric text in the Valid Time field.
  • The Valid Time must be a number only (e.g., 15, 30, 60).
  • Remove any letters, spaces, or special characters from the field.
  • Enter only numeric digits.

"The valid time cannot be a negative number" Error

  • You've entered a negative number (e.g., -5) in the Valid Time field.
  • Valid Time must be 0 or a positive number.
  • Change the value to 0 or a positive number (e.g., 15, 30).

Countdown Not Appearing After User Wins

  • Check if "Display countdown reward reminder" is enabled in the Countdown section.
  • Verify that a Valid Time value has been entered.
  • Make sure you saved your promotion after enabling the countdown.
  • Check if the user actually won a reward (countdown only appears for winning spins).
  • Test the promotion in a different browser or clear browser cache.

Countdown Not Counting Down

  • Check if JavaScript is enabled in the browser.
  • Verify that the countdown is properly enabled and saved in your promotion settings.
  • Check browser console for any JavaScript errors (press F12 to open developer tools).
  • Try refreshing the page or clearing browser cache.
  • Test in a different browser to rule out browser-specific issues.

Countdown Colors Not Changing

  • Make sure you clicked on the color box and selected a color from the color picker.
  • Check that the color boxes are enabled (not grayed out).
  • Try clicking on the color box again and selecting a different color.
  • Refresh the preview to see if the colors update.
  • Make sure you saved your promotion after changing colors.

Countdown Time Resets or Doesn't Persist

  • The countdown uses browser localStorage to persist time. If localStorage is disabled or cleared, the timer may reset.
  • Check if the browser allows localStorage (most modern browsers do by default).
  • If the user clears their browser data, the countdown timer will reset.
  • Each user gets their own independent timer that starts when they win.

Countdown Shows Wrong Time Format

  • The countdown automatically formats based on duration:
    • Less than 1 hour: Shows as "MM:SS" (e.g., "15:00")
    • 1 hour or more: Shows as "HH:MM:SS" (e.g., "01:30:00")
  • This is automatic and cannot be changed manually.
  • The format ensures the countdown is always readable.

Cannot Enable Countdown Toggle

  • Make sure you're logged in and have permission to edit the promotion.
  • Try refreshing the page and attempting again.
  • Check for any JavaScript errors in the browser console (press F12).
  • Ensure you're not in a read-only mode or trial account with limited features.
